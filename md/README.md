## 📝 文檔維護規則

### 新增文檔
- **核心文檔**：重要且長期有效的內容
- **版本記錄**：每次訓練的配置和結果
- **操作指南**：具體操作步驟
- **歷史歸檔**：已過時但保留參考

### 命名規範
- 核心文檔：`功能名稱.md`
- 版本記錄：`vN訓練配置.md` 或 `vN訓練監控指南.md`
- 操作指南：`操作名稱指南.md`

### 更新頻率
- **每次訓練後**：更新 AI交接手冊.md 的實驗日誌
- **重大調整後**：創建新版本記錄
- **發現過時內容**：移至歷史歸檔
